### Control System Requirements (from US Tour specs)

- Computer control AND local control (bypassing processing unit, but not at winch)
- All control located in one area
- Complete backup system for computer crash
- E-stop system for all automated effects
- Position feedback: primary encoder to computer, secondary encoder to local control
- **Positioning accuracy: 1/8" repeatable**
- Conditionals programmable (trigger cue before previous completes)
- Programming in distance (feet/inches) and time (seconds) - *Note: Korea prefers metres*
- Manual control at individual winches as last resort
- Additional monitor at production table during tech

### Deck Track Construction Notes

- Show Deck must be setup and rigged at shop with load test
- Korean or domestic wire rope for drive lines
- "Floating dog" design required (not "skating dog")
- Dogs with set screw attachment to drive line
- Removable lids: flat head machine screws to tapped plate (no wood screws/lags)
